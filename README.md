# Web Scraper 🤖

> Automacao de raspagem de dados publicos com exportacao para CSV e Excel.

## Sobre

Script em Python para automatizar a coleta de dados da web utilizando Selenium 
e BeautifulSoup. Inclui exportacao para formatos estruturados (CSV, Excel), 
agendamento cron e envio de relatorio por e-mail.

## Stack

Python 3 • Selenium • BeautifulSoup • requests • openpyxl • schedule

## Funcionalidades
- Raspagem com navegador automatizado (Selenium)
- Parsing de HTML com BeautifulSoup
- Exportacao para CSV e Excel (openpyxl)
- Agendamento recorrente (schedule)
- Relatorio automatico por e-mail
- Tratamento de erros e retry

## Metrica
Economiza 10 horas por semana de coleta manual de dados.
